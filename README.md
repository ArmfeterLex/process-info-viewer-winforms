Минималистичная программа для получения базовой информации о системе и запущенных процессах.

## Функциональность

### 💻 Информация о системе
- Отображение количества процессоров (ядер)

### 📋 Список процессов
- Получение всех процессов с видимыми окнами (MainWindowHandle != IntPtr.Zero)
- Вывод названий процессов в текстовое поле

## Интерфейс

- Кнопка "butProcCount" — показать количество процессоров
- Кнопка "butProcesses" — показать список процессов с окнами
- Текстовое поле для отображения результатов

## Куда можно встроить

- В системный монитор
- В утилиту для диагностики
- В диспетчер задач с кастомным интерфейсом
- Как пример для обучения работе с классами Process и Environment

## Требования

- Windows
- .NET Framework / .NET Core с WinForms

## Примечание

Программа показывает только процессы с видимыми окнами (фильтр по MainWindowHandle). Для получения всех процессов без исключения можно убрать условие в цикле.

## Примеры использования

```csharp
// Получить количество процессоров
int cpuCount = Environment.ProcessorCount;

// Получить все процессы с окнами
Process[] processes = Process.GetProcesses();
foreach (Process p in processes)
{
    if (p.MainWindowHandle != IntPtr.Zero)
    {
        Console.WriteLine(p.ProcessName);
    }
}
